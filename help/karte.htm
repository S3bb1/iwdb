Karte


Wozu braucht man die Karte?
Die Karte listet alle Systeme innerhalb einer ausgewählten Galaxie auf, die in der Datenbank eingelesen wurden.

Welche Daten müssen eingelesen werden?
Universum → Copy&Paste des Systems in der xml-Ansicht
Im späteren Verlauf kann man auch ganze Galaxien über das Gebäude Universumskontrolleinrichtung (Uni-XML) einlesen
Die Aktualität der Karte ist farblich gekennzeichnet

Wenn man ein System anklickt, werden dessen Planeten in der taktischen Ansicht aufgelistet. Zudem könnt ihr auch direkt ablesen, wann das System zuletzt aktualisiert wurde.
